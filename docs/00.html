¿Qué es la programación declarativa?

Es un paradigma (estilo) de programación en el que el programador/a expresa el resultado esperado de un programa sin indicar cómo calcularlo.

Se centra en qué calcular en lugar de cómo calcularlo

Ejemplo

  • Dada una lista de personas, queremos devolver el nombre de la que tenga mayor estatura, siempre que tenga la edad mínima pasada como parámetro.
  • Si hay empate entre varias personas, se devuelven todas ellas.
  • Si no hay ninguna persona que supere la edad mínima, se devuelve una lista vacía.

Ejemplo: solución imperativa


          fun mayorAltura(personas: List<Persona>, edadMinima: Int): List<String> {
              var elegidos = mutableListOf<String>()
              var alturaMaxima: Int = Integer.MIN_VALUE

              for (p in personas) {
                  if (p.edad >= edadMinima) {
                      if (p.altura > alturaMaxima) {
                          elegidos = mutableListOf(p.nombre)
                          alturaMaxima = p.altura
                      } else if (p.altura == alturaMaxima) {
                          elegidos.add(p.nombre)
                      }
                  }
              }

              return elegidos
          }

Ejemplo: solución declarativa


          fun mayorAltura(personas: List<Persona>, edadMinima: Int) =
              personas.filter { it.edad >= edadMinima }
                  .groupBy { it.altura }
                  .maxByOrNull { it.key } ?.value
                  ?.map { it.nombre } ?: emptyList()
  • Enfoque de más alto nivel
  • Más fácil de entender (si conoces el lenguaje 😅)
  • (A veces) más ineficiente

Paradigmas declarativos

  • Programación funcional (Haskell, Erlang, Elixir, Scala)
    • Los programas se definen mediante funciones «matemáticas».
  • Programación lógica (Prolog)
    • Los programas se definen mediante predicados lógicos.
  • Programación basada en restricciones (MiniZinc)
    • Se especifican las condiciones que ha de cumplir la solución a un determinado problema.
  • Lenguajes específicos de dominio (SQL, XQuery, HTML)
